# KPIs – File Upload Validator

## KPI 1: Response Time
- Definition: Time taken to validate a file input
- Target: < 1 second

## KPI 2: Validation Success Rate
- Definition: Percentage of valid inputs processed correctly
- Target: 100%

## KPI 3: Error Rate
- Definition: Percentage of inputs that result in validation errors
- Target: < 10%

## KPI 4: Deployment Success Rate
- Definition: Percentage of successful CI/CD pipeline runs
- Target: 100%

## KPI 5: Availability
- Definition: Uptime of deployed application
- Target: 99%+
