# Privacy Note – File Upload Validator

## Data Collected
- Filename input
- File size input
- Password (for authentication)

## Purpose of Data Collection
- To validate file properties
- To restrict access to authorized users

## Data Storage
- No data is stored permanently
- All inputs are processed temporarily during session

## Data Retention
- No retention of user data after session ends

## User Rights
- Users can choose not to input data
- No personal identifiable information is collected

## Security Measures
- Password protected access
- Input validation to prevent malicious input
