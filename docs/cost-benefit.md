# Cost–Benefit Analysis – File Upload Validator

## Development Cost

Estimated total development effort:

- Total hours: 70 hours
- Hourly rate: $5/hour

**Development Cost = 70 × 5 = $350**

---

## Operational Cost

- Hosting (Streamlit Cloud): $0/month
- Maintenance effort: 2 hours/month × $5/hour = $10/month

**Total Operational Cost = $10/month**

---

## Tangible Benefits

- Reduces manual file validation time
- Prevents upload of invalid or harmful file types
- Improves efficiency in file handling processes
- Reduces human errors in validation

---

## Intangible Benefits

- Improved user experience
- Better system reliability
- Increased trust in system outputs
- Enhanced security awareness
- Scalable design for future expansion

---

## Return on Investment (ROI)

Assume:
- System saves 1 hour per day for a user
- Value of time: $5/hour

Monthly savings:
- 30 days × $5 = $150/month

### ROI Calculation

ROI = (Benefit − Cost) / Cost

Using 1-month:

- Benefit: $150
- Cost: $350 (initial dev)

ROI = (150 − 350) / 350 = **-57% (first month)**

After 3 months:

- Benefit: $450

ROI = (450 − 350) / 350 = **+28.6%**

---

## Case Study

If this system is used by 5 users, each saving 1 hour per day:

- Daily savings: 5 × $5 = $25
- Monthly savings: $25 × 30 = $750

This significantly increases ROI and makes the system highly cost-effective.

---

## Recommendation

**Proceed**

The system demonstrates strong long-term value despite initial development cost. As usage increases, the return on investment improves significantly. The system is scalable, efficient, and beneficial for environments that require consistent file validation.
